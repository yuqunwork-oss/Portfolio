雙臂智慧機器人視覺抓取系統｜2026 COMPUTEX

2026 COMPUTEX 展示專案，結合 ZED 立體攝影機與 YOLO 物件偵測，將目標物（如可樂罐）的 3D 座標即時轉換至機械手臂基座座標系，取代傳統機械手臂僅能依固定座標作業的限制。我負責軟體系統架構設計、Jetson Thor 環境整合、後端 API 串接、Franka FR3 控制流程與 ROS2 整合、DG-5F 靈巧手 API 與手勢控制、視覺座標資料串接，以及實機測試與問題排除。

紙箱瑕疵辨識系統

結合機械手臂自動拍照與 AI 瑕疵偵測模型，即時判別紙箱表面的髒污（dirt）、破損（damage）與凹凸點（dot），並提供毛邊、瑕疵面積門檻可調式介面，串接產線 PLC 訊號自動觸發檢測。

AI 智慧居家照護與 n8n 警報系統

以攝影機即時偵測居家環境中的人物與危險物件（如剪刀等尖銳物），依距離、持續時間與重疊比例計算危險分數，超過門檻時自動觸發 n8n Webhook 發送警報通知，協助遠端照護者即時掌握居家安全狀況。
